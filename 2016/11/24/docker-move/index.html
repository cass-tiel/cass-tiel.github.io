一次 docker 容器迁移

发表于 2016-11-24   |   分类于 docker

摘要

原环境是购买的国内一家docker云解决方案公司的一套私有环境，搭建在公司的阿里云服务器上。

整套环境包括registry、web管理界面、api接口等；主要使用单机的形式运行，几乎没有对官方docker进行修改，新版本功能较为完善，但需要重新购买，因此决定撤掉这套环境。

由于之前是将容器当做虚拟机的形式使用的，因此迁移过程只迁移以下部分：

  • registry中镜像
  • dockerfile
  • 所有容器（保留原ip）

环境说明

原环境

  • 一台master（web,api,registry,etcd）
  • 四台node
  • ubuntu14.04
